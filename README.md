# Implementacion_Clases_Java

# Proyecto: Clase Product en Java

# Descripción

Este proyecto consiste en la creación de una clase llamada `Product` que representa un producto con los siguientes atributos:

- id
- name
- price
- stock

Se aplican conceptos de Programación Orientada a Objetos (POO) como:

- Encapsulamiento
- Constructores (por defecto y parametrizado)
- Métodos getter y setter
- Sobrescritura del método toString()

El programa crea dos objetos de la clase Product y muestra su información en consola.
